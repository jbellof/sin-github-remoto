CSS (Cascading Style Sheets)

CSS, o Hojas de Estilo en Cascada, es un lenguaje de diseño utilizado para aplicar estilos y presentación a documentos HTML. Su función principal es definir cómo se mostrará el contenido HTML en términos de diseño, formato, colores y disposición en la página.
(Las animaciones y estilos de este sitio fueron realizados con CSS y SASS, por ejemplo las animaciones de la barra de navegacion y el movimoento de la imagen.)

SASS

SASS es un preprocesador de CSS compatible con todas sus versiones. Por lo tanto, se trata de una herramienta utilizada por los desarrolladores web para traducir un código de hojas de estilo no estándar a un código CSS estándar, legible por la mayoría de los navegadores.

3 Formas de aprender CSS

Cursos en Línea Específicos de CSS Plataformas como Codecademy, freeCodeCamp, Udemy, y Coursera ofrecen cursos dedicados a CSS. Estos cursos van desde los fundamentos hasta técnicas avanzadas de estilización web.

Crear Sitios Web Simples: La mejor manera de aprender es aplicar lo que estás estudiando. Comienza creando sitios web simples, como tu propia página personal, un portafolio o un blog. Experimenta con diferentes etiquetas y estilos para comprender su uso en la práctica.
Ejercicios y Desafíos de Estilización: Sitios como CodePen, CSSBattle, y Frontend Mentor ofrecen desafíos de estilización y proyectos a los que puedes enfrentarte para mejorar tus habilidades prácticas en CSS.

Foros y Comunidades en Línea: Únete a comunidades de desarrollo web, como Stack Overflow o Reddit (subreddits como r/webdev o r/learnprogramming). Puedes hacer preguntas, obtener consejos y compartir tu progreso con otros.
Hackatones y Grupos de Estudio:Participa en eventos de desarrollo, hackatones o grupos de estudio donde podrás colaborar con otros, enfrentar desafíos y aprender en un entorno colaborativo. Recuerda, la práctica constante es clave para aprender CSS. Comienza con lo básico, experimenta, comete errores y, sobre todo, diviértete mientras exploras este lenguaje fundamental para el desarrollo web.

Selectores y Propiedades

Los selectores permiten identificar qué elementos HTML serán estilizados. Por ejemplo, h1 seleccionará todos los encabezados de nivel 1 y p seleccionará todos los párrafos. Las propiedades como color, font-size y margin controlan el aspecto de estos elementos.L

Reglas CSS

Las reglas consisten en un selector y uno o más pares clave-valor que especifican los estilos a aplicar. Por ejemplo:
h1 {
color: blue;
font-size: 24px;
}

Cascada y Especificidad

La etiqueta body contiene todo el contenido visible de la página web, como texto, imágenes, enlaces, videos, formularios, entre otros. Todo lo que se muestre en el navegador se coloca dentro de la etiqueta .
